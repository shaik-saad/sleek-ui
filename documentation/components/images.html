Images

SLEEK UI provides images of two style variantions

Responsive Image

To make images responsive use class image-responsive 

                        
                            <img class="image-responsive" src="../../assets/image-1.jpg" alt="a cute cat image">

Round Image

To make images round use class image-round 

                        
                            <img class="image-responsive image-round" src="../../assets/image-3.jpg" alt="a cute cat image">
